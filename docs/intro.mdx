# Prerequesites

Let's make the Movie Web Backend on **Google Cloud Run and Vercel**!
> (*Vercel is not necessary if you have a postgrsql database anyway*)

### What you'll need

- [Google Cloud Run](https://cloud.google.com/run)
  - If you do not have it you please visit the link now and register, you will need a:
    - Google Account
    - A Credit/Debit card (Used to verify you, there is no charge)



:::note

**These are not the [official Movie Web docs](https://movie-web.github.io/docs/)**.

:::

:::info

This method currently **supports the Movie Web backend [version 1.3.1](https://github.com/movie-web/backend/pkgs/container/backend/171215961?tag=1.3.1)**.

This is the latest version at the time this website was created. If you want a later version **please file a [new issue](https://github.com/TecEash1/movieweb-backendocs/issues/new)** or **[make the docker file yourself](./docker)**.

:::
